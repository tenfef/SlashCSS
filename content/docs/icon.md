##### Basic Usage

The icon single is defined by the class `icon` this can be used in conjunction with any icon vendor syntax.

{{% codeblock key="language" definition="html" margin="bottom" %}}
```html
<i class="icon fas fa-star "></i>
```
{{% /codeblock %}}
